HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Elements nested inside other elements are called child elements
  • The element that holds all of the children is called parent element
  • Element is a type of content on a webpage
  • HTML is used to build the frame of a webpage
  • HTML attributes are always defined in the beginning element, never the closing element

CSS

  • CSS is used to style and layout web pages — for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • css style must be linked in HTML to show on webpage

GIT

  • git add -A: adds all files
  • git commit -m: adds commit message
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it

JavaScript

  • JavaScript is a powerful programming language that can add interactivity to a website.
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which the computer executes code in a script
  • JavaScript is case sensitive
  • String- To signify that the value is a string, enclose it in single or double quote marks.
  • Numbers dont have quotes around them
  • Boolean- This is a True/False value. The words true and false are special keywords that don't need quote marks.
  • Array- This is a structure that allows you to store multiple values in a single reference.
  • object- This can be anything. Everything in JavaScript is an object and can be stored in a variable. Keep this in mind as you learn.
  • An operator is a mathematical symbol that produces a result based on two values (or variables).
  • Conditionals are code structures used to test if an expression returns true or not. A very common form of conditionals is the if...else
